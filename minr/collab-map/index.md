# Minr Collaborations Map

An interactive network visualization showing the collaborations between members of the Minr community. This tool allows you to explore connections between creators and search for specific community members.

The page features:
- An interactive graph visualization of the collaboration network
- Search functionality to find specific creators
- Visual representation of community connections and relationships

This is a custom interactive application with a specialized visualization interface.
