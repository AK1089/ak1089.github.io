Some Software I Like Using

This is a pretty incomplete list of software I think is extremely goodI use all of these pretty regularly, and I think those that are paid are well worth the money.. I've tried to stay away from social-media style uses and just "nice websites" - these are mostly utility-based tools. The idea is to list pieces of software which have unusually good user experiences, effectiveness, or features you can't get anywhere else.

  • The Browser Company's Arc - A browser made by people with attention to detail who care about user experience
  • Anthropic's Claude 3 Opus - AI assistant with fantastic design which is smart and personable
  • Last.fm - Tracks every song you listen to and displays summary statistics
  • Ground News - Groups news articles from many sources with bias ratings for each
  • TEXMaker - A powerful LaTeX editor with an integrated PDF viewer
  • KeyBR - Website which allows you to practice typing and improve your speed
  • Excalidraw - Online drawing tool with useful features and a nice aesthetic
  • Fusion 360 - Easy and powerful CAD software for designing 3D prints
  • Cold Turkey Blocker - The most effective website and distraction blocker I've used
  • One Sec - An app blocker for phones which makes you wait before opening apps
  • Alarmy - Alarm which forces you to complete various tasks in order to turn it off
