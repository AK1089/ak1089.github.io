Some Software I Like Using

This is a pretty incomplete list of software I think is extremely goodI use all of these pretty regularly, and I think those that are paid are well worth the money.. I've tried to stay away from social-media style uses and just "nice websites" - these are mostly utility-based tools. You'll probably know most of these. They're not in any particular order.

  • Opera GX - Browser with control over RAM limits, cache usage, and a helpful sidebar
  • OpenAI's GPT-4 - AI assistant which is smart and good with technical stuff
  • Anthropic's Claude 3 Opus - AI assistant which is much more personable and friendly
  • Spotify - Enables access to pretty much the entire human library of music available to stream
  • Last.fm - Tracks every song you listen to and displays summary statistics
  • Ground News - Groups news articles from many sources with bias ratings for each
  • InoReader - Simple RSS reader with good feed organisation and control
  • GitHub - Keeps all your code safe, tracks all edits to it, hosts this site, and helps collaboration
  • Visual Studio Code - My favourite IDE for writing code of pretty much any language
  • TEXMaker - A powerful LaTeX editor with integrated PDF viewer
  • KeyBR - Website which allows you to practice typing and improve your speed
  • Google Sheets - In-browser spreadsheet with most powerful features and easy formatting
  • Excalidraw - Online drawing tool with useful features and a nice aesthetic
  • Dingboard - Fast AI-powered easy image editing within the browser
  • Fusion 360 - Easy and powerful CAD software for designing 3D prints
  • Cold Turkey Blocker - The most effective website and distraction blocker I've used
  • One Sec - An app blocker for phones which makes you wait before opening apps
  • Forfeit - An app which forces you to commit to things under monetary penalty
  • Alarmy - Alarm which forces you to complete various tasks in order to turn it off
