Software I Like

published December 24, 2024

This is a pretty incomplete list of software I think is extremely good. I use all of these pretty regularly, and I think those that are paid are well worth the money. I’ve tried to stay away from social-media style uses and just “nice websites” - these are mostly utility-based tools. The idea is to list pieces of software which have unusually good user experiences, effectiveness, or features you can’t get anywhere else.

  1. The Browser Company’s Arc - A modern browser with neat organisation and good aesthetics
  2. Last.fm - A site which tracks every song you listen to with summary statistics and a free API
  3. Texifier - The best LaTeX editor I’ve ever used, with live typesetting and a customisable layout
  4. KeyBR - A website for typing speed practice which is effective at training and improvement
  5. Excalidraw - A free whiteboard with many useful features for diagrams and lovely aesthetics
  6. Autodesk Fusion 360 - Easy and powerful CAD software for designing 3D models and prints
